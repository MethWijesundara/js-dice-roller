# ✨ Dice Roller ✨

A simple interactive dice roller. 

## Features
- Roll multiple dice based on user input
- Generates random numbers from 1 to 6
- Displays results dynamically on the page

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
The user enters the number of dice to roll.
The program generates random values using JavaScript and displays the results on the screen.

## Purpose
This project was built to practice DOM manipulation, loops, and random number generation in JavaScript.
